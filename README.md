# CodeAlpha Responsive Image Gallery
A Responsive Image Gallery with Lightbox, Navigation Controls, and Hover Effects built using HTML,CSS,and JavaScript 

## Features 
- Responsive Design
- Lightbox Preview
- Next / Previous Buttons
- Hover Effects 
- Smooth Animations

## Technologies Used
- HTML5
- CSS3
- JavaScript 

## Project Structure 
```
index.html
style.css
script.js
images/
```
